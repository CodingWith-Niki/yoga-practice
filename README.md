# 🧘‍♀️ Yoga & Wellness Website

Welcome to the **Yoga & Wellness** project! This is a clean, modern, and responsive website designed to promote physical fitness and mental well-being through Yoga.

## 🌟 Features
- **Calm UI:** A peaceful and eye-soothing design specifically for wellness.
- **Navigation:** Smooth navigation between Home, About, and Yoga Asanas sections.
- **Interactive Layout:** Well-structured sections for different yoga poses and health tips.

## 🛠️ Tech Stack Used
- **HTML5:** For the core structure of the website.
- **CSS3:** For styling, layouts, and responsive media queries.
- **Google Fonts:** For clean and professional typography.

## 📂 Project Structure
- `index.html` - Main landing page.
- `CSS/` - Folder containing the stylesheet.
- `Images/` - Contains all yoga-related images and icons.
